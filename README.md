# KIZUNA 🐾

Centro de adopción de mascotas.

Proyecto web desarrollado para promover la adopción responsable de animales y conectar mascotas con familias que desean brindarles un hogar.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5.3
- Font Awesome
- Google Fonts
- AOS Animation

## Páginas del sitio

- Inicio
- Mascotas
- Adopta
- Donaciones
- Contacto

## Características

- Diseño responsive para dispositivos móviles.
- Menú de navegación con 5 secciones.
- Animaciones mediante AOS.
- Tarjetas de mascotas.
- Formulario de adopción y contacto.
